Set 集合

Set 接口 (无序,元素不能重复)，继承了 Collection 接口无特有方法。

Set 已实现类与特性：

  • HashSet：哈希表结构，保证元素唯一性依靠 hashCode() 和 equals() 方法(hash 值相同时再判断 equals,相同将不再存储)，不同步
  • TreeSet：是链表结构(指针)，需保证元素唯一性，可排序
  • LinkedHashSet：HashSet 的子类，按存入顺序使用
